# GhostBusters Project

## Stack 

🌱 **Language**: Java 21   
🧩 **Testing**: JUnit

## Description

This project is an application to manage the capture and release of ghosts. It uses Java 21 with the Spring Boot framework for development, and JUnit for unit testing.

## Requirements

- JDK 21 or higher
- Maven or Gradle for dependency management
